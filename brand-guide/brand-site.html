05 — Motion & Animation

Motion Principles

Motion should be purposeful and subtle. Every animation has a reason — reveal, guide attention, or confirm an action. Nothing spins, bounces, or flashes.

Approved Patterns
Fade Up
opacity 0→1 + translateY 20px→0 · 500ms ease
Scale In
scale .9→1 + opacity · 400ms ease
Slide In
translateX -20px→0 + opacity · 450ms ease
Motion Rules
Duration 100–600ms. Fast for micro, 400–600ms for reveals.
Easing ease or cubic-bezier(.4,0,.2,1). Never linear for UI.
Hover translateY(-2px to -4px) + box-shadow. 200ms ease.
Scroll Fade-up on enter only. No continuous scroll animations.
Never Spin, bounce, flash, rotate, infinite loops (except loading states).

Respect prefers-reduced-motion. All non-essential animations must be disabled when the user has reduced motion enabled in their OS.

07 — Voice & Tone

How We Sound

DemandScience speaks in a clear, direct, human voice. We simplify B2B complexity without losing precision. We never use jargon when plain language works.

"Empowering B2B Marketers to take back control of their growth."

Elevator Pitch — Short

DemandScience helps B2B companies grow faster by connecting them with the right buyers at the right time. We provide demand generation solutions that are data-driven, scalable, and built for revenue impact.

08 — Do & Don't

Brand Do's & Don'ts

These rules exist because past violations damaged brand credibility. When in doubt, left-align it and add more whitespace.

check_circle Do

  • Use white backgrounds on all slides
  • Left-align all slide text
  • Use Poppins Bold for all headings
  • Use CTA Red (#D42F5B) only on buttons
  • Use Google Material Symbols Rounded icons
  • Add generous whitespace — if it feels tight, add more
  • Use authentic, candid photography (no posed portraits)
  • Follow the approved chart color order: Navy → Blue → Pink → Grey
  • Keep charts on white backgrounds with minimal gridlines
  • Get Design approval before creating any external asset

cancel Don't

  • Use gradient or navy as slide backgrounds
  • Center body text on slides
  • Use Montserrat, Lato, Open Sans, or any other font
  • Use CTA Red decoratively or as a background
  • Use Font Awesome, Heroicons, or Feather icons
  • Cram content — one focal point per slide/screen
  • Use stock photography with subjects looking at camera
  • Use default Excel/Sheets colors in charts
  • Create public-facing assets without an approved template
  • Describe any external request as "quick" or "informal"

Brand Review Required?

Any external asset, new logo treatment, co-brand, or off-system design must be reviewed by the Design team before publishing.

09 — Brand Governance

Enforcement System

Brand governance exists to protect DemandScience's market presence and ensure every customer touchpoint reflects the same quality standard. There are no exceptions to STOP rules.

🚫 STOP — Halt All Work

  • Icon library other than Material Symbols Rounded
  • Font other than Poppins or Inter
  • Off-palette color requested or used
  • New logo treatment or variant
  • Public asset without approved template
  • Co-brand or partner lockup
  • Request described as "quick external"
  • Charts with default Excel/Sheets colors

⚠ WARNING — Flag & Fix

  • Old hex values used (#05195F, #0066FC)
  • Italic used for emphasis (use bold)
  • Justified paragraph text
  • Heavy card shadow (> spec)
  • Sharp-cornered cards (missing 10px radius)
  • Navy/blue CTA button (not pink)
  • Missing alt text on images
  • Gradient on non-approved sections

✅ Approved — No Review

  • Internal presentations (approved template)
  • Client decks (approved template, no new layouts)
  • One-pagers using approved doc template
  • Charts with approved color order on white
  • Icons from Material Symbols Rounded
  • Email using approved template
  • Web pages using component library
  • Internal Slack/comms graphics
