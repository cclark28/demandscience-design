# Google Slides Templates

Drop reference decks and approved template files here.

Claude will read these files to understand slide structure, layout patterns, and approved design decisions before generating new slide content.

## How to add a reference deck

1. Export the deck as PDF or screenshot individual slides as PNG/JPG
2. Name files clearly: `deck-name_slide-type.png` (e.g., `q1-report_cover.png`)
3. Add a note in `/channel-guides/google-slides.md` under **Reference Decks** describing what it demonstrates
