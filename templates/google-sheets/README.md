# Google Sheets — Brand Reference Files

Three CSV files ready to import into Google Sheets as separate tabs.

## How to import into Google Sheets

1. Go to **sheets.google.com** → create a new spreadsheet
2. **File → Import → Upload** → select `ds-colors.csv`
3. Choose "Insert new sheet(s)" → Import
4. Repeat for `ds-fonts.csv` and `ds-icons.csv`
5. Rename each tab: **Colors**, **Fonts**, **Icons**
6. On the Colors tab: select the HEX column → apply background colors manually for a visual swatch view
7. Share the spreadsheet company-wide with **View** access

## Files

| File | Contents |
|------|----------|
| `ds-colors.csv` | Full color palette · hex values · RGB · CSS variables · usage rules |
| `ds-fonts.csv` | Font families · slide/web/doc type scales · formatting rules |
| `ds-icons.csv` | Icon library details · approved icon list · sizing guide · usage rules |
