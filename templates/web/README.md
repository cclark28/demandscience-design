# Web Templates

Drop reference screenshots, HTML starters, and approved web page examples here.

Claude will read these to understand layout patterns and design decisions for web and landing pages.

## How to add a reference

1. Screenshot full pages or key sections as PNG
2. Name clearly: `page-type_section.png` (e.g., `landing-page_hero.png`)
3. Add a note in `/channel-guides/web.md` under **Reference Pages**
