# Web & Landing Pages Channel Guide

Reference `ds-brand-design.md` for all base rules. This file covers web-specific visual application.
For SEO requirements — meta tags, structured data, performance, headings, images — see `channel-guides/seo.md`. Both files apply to every web page.

---

## Page Structure

- Default background: white `#FFFFFF` everywhere.
- Gradient only in: hero band, section divider bands, closing CTA band.
- Sections stack vertically with consistent spacing.

---

## Hero Section

```
[ Nav ]
[ HERO — white or gradient background ]
  Left column (55%):
    - Eyebrow label (optional): Inter 14px medium, Blue
    - H1: Poppins 50px/60px Bold, Navy
    - Body: Inter 18px/28px, Dark Grey
    - Primary CTA button (Red Orange `#F40356` pill) + optional secondary (blue outline pill)
  Right column (45%):
    - Product UI mockup, illustration, or key visual
    - Rounded rectangle mask, 10px radius
```

---

## Section Patterns

**Standard content section (white):**
- Max content width: 1200px, centered
- Section padding: 96px top/bottom
- Heading: H2, left-aligned (or centered for short, punchy sections)
- Body below heading

**Card grid section:**
- 2 or 3 columns
- Cards: white or light grey background, 10px radius, subtle shadow
- Consistent gap: 24–32px

**Stat strip:**
- White background
- 3–4 stats, horizontally distributed
- Large number: Poppins SemiBold, 48px+, Navy or Blue
- Label: Inter, 16px, Medium Grey

**Section band (gradient or navy):**
- Full-width
- Short headline + subhead, centered, white
- Optional: single CTA button (Electric Blue, pill)
- Use sparingly — maximum 1–2 per page

---

## Navigation

- Background: white with subtle bottom border or soft shadow on scroll
- Logo: left-aligned
- Nav links: Inter 15px medium, Dark Grey; hover → Blue
- Primary CTA in nav: Red Orange `#F40356` pill button
- Mobile: hamburger → slide-in drawer

---

## Footer

- Background: Navy `#061947`
- Text: white / medium grey
- Logo: white version
- Links: white, underline on hover
- Divider: subtle opacity line

---

## Responsive Breakpoints

| Breakpoint | Width      | Notes                              |
|------------|------------|------------------------------------|
| Mobile     | < 768px    | Single column, stacked layout      |
| Tablet     | 768–1024px | 2-column where applicable          |
| Desktop    | > 1024px   | Full layout, max-width 1200px      |
